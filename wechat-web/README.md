# Wechat Mini Program (uni-app x)

## 简介

使用 uni-app x 实现微信小程序端：登录、搜索栏、任务中心、我的任务、创作助手、创作学院、奖励模块。

## 开发

1. 使用 HBuilderX 或 `@dcloudio/uni-app` 工具打开 `wechat-web`
2. 配置小程序 AppID，填写到 `manifest.json` 的 `mp-weixin.appid`
3. 运行到微信开发者工具调试

## 运行环境变量

- 在 `manifest.json` 的 `networkTimeout.request` 以及 `BASE_URL` 中配置后端地址
